###############
# Description #
###############

Free python scripts, you can use as you want.
Words.py - script, which generates all possible words from sequence of letters
in input file with set length in it. Works with many languages and encoding.
Really, it gets all words with that length from vocabulary file (you can
download any vocabulary you want) and check if it can be constructed from that
sequence of letters.

##############
# How to use #
##############

1. You have to install python chardet package.
2. Download vocabulary, where each new word is in new line.
3. Run words.py and you will get all possible words in the output file.
All filenames are set in the config.cfg
